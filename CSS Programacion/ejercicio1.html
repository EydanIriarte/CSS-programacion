Esto es una prueba de colores RGB y HEX

RGB se usa para cualquier diseño digital, este permite crear distintos tonos de colores usando los colores primarios (rojo,verde,azul)

Los colores hexadecimales permiten representar matices precisos de rojo, verde y azul mediante la combinación de 16 valores distintos para cada componente,ofreciendo así una amplia gama de tonalidades para cada color básico.
